Routines to build the linear system

pyeq.lib.forward_model.build1.build1(lGts, np_dates, NAME_OBS, GREEN, UP, s_up, date_tol=0.1, verbose=True, debug=False)[source]¶
Parameters

  • lGts – list of Gts

  • np_dates – numpy 1D array of dates

  • NAME_OBS – numpy 1D array of site name

  • GREEN – Green tensor as a 4D numpy array

  • UP – boolean, True means up component will be included

  • date_tol – tolerance (unit in fractional day) on dates to search for dates in time series corresponding to the modeling period. Use 0.1 days for daily time series.

  • verbose – verbose mode

Return ATPA, ATPB

LHS matrix and RHS vector ready for inversion algorithm
