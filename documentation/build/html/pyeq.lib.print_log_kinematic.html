pyeq.lib.print_log_kinematic.interp_slip_at_dates(SLIP, slip_dates, dates)[source]

Given a slip increment provided a a 1D vector, with dates slip_dates, calculates the slip predicted at dates using linear interpolation

Parameters:

  • SLIP – a 1D numpy vector of slip increments with nstep x nfaults size

  • slip_dates – a 1D numpy vector of slip dates of size nstep + 1

  • dates – a 1D numpy vector with interpolation dates

Returns:

a 2D numpy vector of n_dates as row and nfault as columns

pyeq.lib.print_log_kinematic.predict_disp_from_model(SLIP, G)[source]

Given a slip model provided as a 1D vector, with Greens function G, returns the displacement prediction

np.dot(G,SLIP) So the result is a vector with size G.shape[0] if SLIP is a matrix with SLIP at different times ordered by columns returns np.matmul(G,SLIP)

Parameters:

  • SLIP – a 1D or 2D numpy array of slip

  • G – Grren’s function matrix

Returns:

a numpy array
