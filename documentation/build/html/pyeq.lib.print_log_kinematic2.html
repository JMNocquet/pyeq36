pyeq.lib.print_log_kinematic2.interp_slip_at_dates(SLIP, slip_dates, dates)[source]¶

Given a slip increment provided a a 1D vector, with dates slip_dates, calculates the slip predicted at dates using linear interpolation

Parameters

  • SLIP – a 1D numpy vector of slip increments with nstep x nfaults size

  • slip_dates – a 1D numpy vector of slip dates of size nstep + 1

  • dates – a 1D numpy vector with interpolation dates

Returns

a 2D numpy vector of n_dates as row and nfault as columns
