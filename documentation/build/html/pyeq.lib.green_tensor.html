pyeq.lib.green_tensor module

Manipulate the Green tensor

pyeq.lib.green_tensor.GREEN_ENU_TO_GREEN_RAKE_MAIN_RAKE_CONJUGATE(GREEN, GEOMETRY, SGEOMETRY, rake_type, rake_or_pole)[source]

GREEN IS A TENSOR OF DIM 4 The input GREEN TENSOR is GREEN(i,j,k,l) is the prediction for dislocation i at site j component k for rake l k=0,1,2 = east, north, up l=0,1 : rake_00 & rake_90 I want for this code NEW_GREEN(i,j,k,l) is the prediction for dislocation j at site i component k for rake l k=0,1,2 = east, north, up l=0,1 : rake_principal & rake_conjugate

the rake can be defined either by an euler pole and geometry (providing position strike and dip to calculate the relative slip) or a fixed rake
