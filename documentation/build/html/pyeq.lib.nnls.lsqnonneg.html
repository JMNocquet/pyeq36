pyeq.lib.nnls.lsqnonneg module¶

A Python implementation of NNLS algorithm

References: [1] Lawson, C.L. and R.J. Hanson, Solving Least-Squares Problems, Prentice-Hall, Chapter 23, p. 161, 1974.

Contributed by Klaus Schuch (schuch@igi.tugraz.at) based on MATLAB’s lsqnonneg function

pyeq.lib.nnls.lsqnonneg.lsqnonneg(C, d, x0=None, tol=None, itmax_factor=3)[source]¶

Linear least squares with nonnegativity constraints.

(x, resnorm, residual) = lsqnonneg(C,d) returns the vector x that minimizes norm(d-C*x) subject to x >= 0, C and d must be real
